### Analyzing website traffic of a website

This project is done to demonstrate SQL data analysis skills. 
There are two tables used in this analysis:
	-	website_sessions
	-	orders
	
	-	columns in website_sessions
			-	session_id
			-	created_at
			-	user_id
			-	repeat_session_id
			-	utm_source, 
			-	utm_campaign
			-	http_referer
			-	device_type
			
	-	columns in orders
			-	order_id
			-	session_id
			-	created_at
			-	user_id
			-	primary_product_id
			-	items_purchased
			-	price_usd
			-	cogs_usd
